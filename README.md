# Gestor-Estoque-V2
Retomando a ideia do projeto gestor de estoque realizado anteriormente serão feitas mudanças e adpatações para atender a novas demandas, inclusive a criação de um sistema web equivalente ao sistema desktop.

Projeto: Gestor de estoque v2.0.0 
Status: Em desenvolvimento 
Tecnologias: Java, MySQL, HTML, CSS 
Desenvolvedores: Érick Neujahr Pimentel 
Objetivo: Projeto desenvolvido visando o controle e gerenciamento de estoques. 
Funcionalidades: Cadastro de produtos, cadastro de fornecedores, registro de movimentações, acesso a relatorios e gestão de usuários.
